# 在线学习平台移动Web客户端

## 技术栈
```
vue2 + vuex + vue-router
vant + webpack + ES6/7 + axios
svg + less
```

### 项目运行
#### 注意：由于涉及大量的 ES6/7 等新属性，node 需要 6.0 以上版本
```
git clone https://github.com/MemoryDewey/online-study-mobile-web
cd online-study-mobile-web
npm install
npm run server
```
